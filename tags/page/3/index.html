What This Is All About

You might have stumbled upon this site by randomly browsing the internet, or following a link from one of the mixes accompanying this site and be wondering: “ok, so what this is all about?” ‘Decaying Foundations’ is a fusion of fictional stories and electronic music used to create and shape the alternative world, which resembles ours so much. A reflections from the (im)possible future, if you will. It’s about the world and its everyday dystopias we’re experiencing right now.
